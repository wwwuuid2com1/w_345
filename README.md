<p>安装说明：
1.Linux服务器解压覆盖到/www/server/panel/BTPanel/   Windows服务器解压覆盖到宝塔安装的磁盘位置\BtSoft\panel\BTPanel
2:(1)Linux进入SSH，输入bt，选择9，清空面板缓存
(2)Windows按住Windows键和“R”键，输入cmd，再输入bt，选择数字9，清空面板缓存
3.清空浏览器缓存
4.刷新浏览器即可<p>
